# Defect Tracker

### Project for SEPM by Team Broski

> Deployed at https://bugtracker-sepm.herokuapp.com/

## Technology Stack and Dependencies

- **Front-end**
  - HTML
  - CSS
  - BootStrap
  - Firebase
  - EJS

- **Back-end**
  - Node.js
  - Express.js
  - MongoDB
